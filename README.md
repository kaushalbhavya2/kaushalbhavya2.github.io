## Tech

Single `index.html` file. All CSS and JavaScript are inlined, so the site loads instantly and is easy to edit or fork. Responsive down to mobile, with scroll-triggered animations and a dark hero section.

## Updating the site

To make changes, edit `index.html` directly in this repo (pencil icon on the file view) or upload a new version via **Add file → Upload files**. GitHub Pages automatically redeploys within a minute or two.
